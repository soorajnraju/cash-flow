## 🔧 Installation & Development

```bash
# Clone the repository
git clone https://github.com/soorajnraju/cash-flow.git

# Navigate to project directory
cd cash-flow

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
